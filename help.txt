NOVIUS BLOCS

PREVIEW DANS LES ENHANCERS

- il est possible d'ajouter la preview des blocs lors de l'ajout dans un enhancer. Pour ce faire, il est nécessaire d'étendre la config de l'appli, ici :
    controller/admin/bloc/enhancer.config.php

    et de set la propriété 'custom' à true dans l'array preview : exemple de fichier étendu :

return array(
    'preview' => array(
        'custom' => true,
    ),
);

    l'application se servira par défaut de sa vue preview personnalisé, il vous suffira de gérer les possibles collisions css.
    si vous souhaitez utiliser une vue perso, il faudra rajouter la clé 'view' dans le même tableau que 'custom'

    Pour gérer des CSS propres à la preview, il suffit de créer, dans le même répertoire que les css pour l'admin, des fichiers de type : nom_bloc.preview.css

    (donc dans public/static/apps/css/blocs/admin) et de configurer votre css comme vous l'avez fait pour l'affichage en back.
